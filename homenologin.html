Programmer is a profession occupied by people who create
programs based on software engineering specification sheets.
When creating a program, there is a flowchart that must be
considered and then translated into computer language
using certain code so that it becomes a command to
the computer to carry out certain tasks.

Programming languages, such as CSS, SQL, HTML, JavaScript,
PHP, XML, Python, and so on have become familiar things
for a programmer. Now, when the program is finished, the
programmer must make sure that the program is operating
properly. The programmer will perform various tests by
predicting each case that will occur because the computer
will not be able to operate when there is an error. Therefore,
programmers must be error-friendly. Because every time
you create a program, there is always trial and error.
